# MACHINA（タスク自動生成）仕様

## 概要

Slack / Discord チャンネルを監視し、メッセージからタスクを自動生成する M3 モジュール。ルールベースのテキスト解析（将来: Claude Haiku AI 解析）でタイトル・優先度・アサイン・納期を自動設定する。

## 機能要件

### チャンネル監視

- グループ別に複数チャンネルを監視可能
- プラットフォーム: Slack / Discord
- Webhook 受信で新規メッセージを検出

### テキスト解析

- ルールベースのキーワード検出
- タスクタイトル・優先度・アサイン・納期の自動抽出
- AI 解析時の信頼度スコア (0.0〜1.0)

### タスク自動生成

- 生成元: `auto` (自動検出) / `command` (コマンド) / `manual` (手動)
- ステータス: `pending` / `in_progress` / `done` / `cancelled`
- 完了キーワード検出による自動ステータス更新

### PM リレー

- M2（PM）へのアダプタパターンによるリレー
- リレー済みフラグ・PM タスク ID の管理
- クリティカルパスフラグの設定
