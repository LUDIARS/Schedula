# MACHINA ユースケース

## UC-MCH-01: チャンネル監視設定

- **アクター**: グループ管理者
- **主フロー**:
  1. 監視対象のプラットフォーム・チャンネルを指定する
  2. Webhook エンドポイントを紐づける
  3. 監視を有効化する

## UC-MCH-02: タスク自動生成

- **アクター**: システム
- **事前条件**: チャンネル監視が有効
- **主フロー**:
  1. Webhook でメッセージを受信する
  2. テキスト解析でタスク情報を抽出する
  3. タスクを自動生成する
  4. タスクログに記録する

## UC-MCH-03: タスク管理

- **アクター**: グループメンバー
- **主フロー**:
  1. 自動生成されたタスク一覧を確認する
  2. タスクのステータス・優先度・アサインを変更する
  3. 変更履歴がログに記録される

## UC-MCH-04: PM リレー

- **アクター**: グループ管理者
- **主フロー**:
  1. タスクを選択して PM にリレーする
  2. PM 側にタスクが作成される
  3. リレー済みフラグが設定される

## UC-MCH-05: 自動ステータス更新

- **アクター**: システム
- **主フロー**:
  1. チャンネルメッセージから完了キーワードを検出する
  2. 対応するタスクのステータスを自動更新する
  3. タスクログに変更理由を記録する
