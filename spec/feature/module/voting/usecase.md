# Voting ユースケース

## UC-VOT-01: 日程調整イベント作成

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. タイトル・説明を入力する
  2. 候補日時を複数登録する
  3. 回答期限を設定する
  4. イベントを作成する

## UC-VOT-02: 投票回答

- **アクター**: 招待されたユーザー
- **主フロー**:
  1. イベントの候補日時を確認する
  2. 各候補に ○/△/× で回答する
  3. コメントを任意で追加する

## UC-VOT-03: 自動回答

- **アクター**: システム
- **主フロー**:
  1. ユーザーのカレンダー空き状況を取得する
  2. 各候補日時と照合する
  3. 空き → ○、一部空き → △、予定あり → × で自動回答する
  4. `isAutoReply = true` で保存する

## UC-VOT-04: 集計結果確認

- **アクター**: イベント作成者
- **主フロー**:
  1. 各候補日時の ○/△/× 集計を確認する
  2. 最適な日程を決定する
  3. イベントを close する
