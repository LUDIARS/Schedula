# Notification ユースケース

## UC-NTF-01: Webhook エンドポイント登録

- **アクター**: 管理者
- **主フロー**:
  1. URL・対象イベント・プラットフォームを指定する
  2. 送信方法（webhook / bot）を選択する
  3. Bot の場合はトークン・チャンネル ID を設定する
  4. エンドポイントを登録する

## UC-NTF-02: 通知設定

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. チャンネルごとに有効イベントを選択する
  2. リマインダーの種類・タイミングを設定する
  3. quiet hours を設定する

## UC-NTF-03: イベント通知配信

- **アクター**: システム
- **主フロー**:
  1. イベントバスでイベントを受信する
  2. 対象ユーザーの通知設定を確認する
  3. テンプレートを適用して通知文を生成する
  4. 各チャンネルに配信する
  5. 配信ログを記録する

## UC-NTF-04: テンプレート管理

- **アクター**: 管理者
- **主フロー**:
  1. イベント名・プラットフォームを指定する
  2. タイトル・本文テンプレートを入力する
  3. テンプレートを保存する

## UC-NTF-05: 通知履歴閲覧

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. 自分宛の通知一覧を取得する
  2. 既読/未読を管理する
