# Smart Scheduler ユースケース

## UC-SS-01: スケジューリングタスク登録

- **アクター**: グループメンバー
- **主フロー**:
  1. グループを選択する
  2. タスク名・所要コマ数・優先度を入力する
  3. 希望曜日・希望コマを任意で設定する
  4. 担当講師を任意で指定する
  5. タスクを登録する

## UC-SS-02: 自動配置の実行

- **アクター**: グループ管理者
- **事前条件**: スケジューリングタスクが1つ以上登録済み
- **主フロー**:
  1. グループの空き状況を計算する
  2. DP ソルバーで最適配置を算出する
  3. 配置結果をスコア付きで提示する
- **代替フロー**:
  - 配置不可のタスクがある場合、failed ステータスで通知する

## UC-SS-03: 配置結果の確定

- **アクター**: グループ管理者
- **主フロー**:
  1. 配置提案を確認する
  2. 確定操作を行う
  3. グループ予定として自動登録される

## UC-SS-04: 配置結果の却下

- **アクター**: グループ管理者
- **主フロー**:
  1. 配置提案を確認する
  2. 却下操作を行う
  3. タスクは未配置状態に戻る
