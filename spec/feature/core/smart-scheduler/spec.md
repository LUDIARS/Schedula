# Smart Scheduler（オートスケジューラ）仕様

## 概要

グループメンバーの空き状況を分析し、DP（動的計画法）ベースのバックトラッキングで最適な予定配置を自動提案するコアモジュール。

## 機能要件

### 空き状況計算

- グループメンバー全員の個人予定・グループ予定・Google Calendar を集約
- 70% 以上のメンバーが空いているコマを「空きスロット」と判定
- 休日・グループイベントも考慮

### DP ソルバー

- 複数のスケジューリングタスクを同時に最適配置
- 各タスクに希望曜日・希望コマ・優先度を設定可能
- 講師の空き時間制約にも対応
- バックトラッキングで制約充足を保証

### 配置結果

- `draft` (提案中) → `confirmed` (確定) → `rejected` (却下) のライフサイクル
- 確定時にグループ予定として自動登録
