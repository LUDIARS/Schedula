# MyPlan（マイプラン）仕様

## 概要

週間ルーティーンを定義し、個人予定を自動生成するコアモジュール。基本パターンと特別パターンを持ち、特別パターンが優先される。

## 機能要件

### パターン優先度

- **basic**（基本パターン）: デフォルトの週間スケジュール
- **special**（特別パターン）: 特定期間に適用される優先スケジュール
- 同一期間に複数パターンがある場合、priority 値が大きいものを優先
- 適用期間が設定されている場合、期間外は無効扱い

### 週間スケジュール形式

```json
{
  "0": [{ "startTime": "09:30", "endTime": "12:30", "title": "授業" }],
  "1": [{ "startTime": "13:30", "endTime": "15:30", "title": "自習" }]
}
```

- キー: 曜日インデックス（0=月〜6=日）
- 値: その曜日のスケジュール配列

### Plan（レガシー）との関係

- plans テーブルはコマベースの繰り返し予定生成に使用
- myPlans テーブルは時間ベースの週間ルーティーンとして導入
- 両方を併用可能だが、myPlans が推奨
