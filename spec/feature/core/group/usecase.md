# Group ユースケース

## UC-GRP-01: グループ作成

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. グループ名・説明を入力する
  2. グループを作成し、作成者を owner ロールで登録する
- **事後条件**: グループが作成され、作成者が owner として所属する

## UC-GRP-02: メンバー招待・追加

- **アクター**: グループ owner / admin
- **主フロー**:
  1. 追加するユーザーを選択する
  2. メンバーロール（admin / member）を指定する
  3. グループに追加する
- **制約**: 同一ユーザーの重複追加は不可

## UC-GRP-03: メンバー削除

- **アクター**: グループ owner / admin
- **主フロー**:
  1. 削除対象のメンバーを選択する
  2. グループから除外する
- **制約**: owner は自身を削除できない（別の owner を指名する必要がある）

## UC-GRP-04: グループ予定登録（曜日ベース）

- **アクター**: グループ owner / admin
- **主フロー**:
  1. タイトル・曜日・コマ・期間を入力する
  2. recurring（毎週繰り返し）または oneshot（特定日のみ）を選択する
  3. グループ予定として登録する
- **補足**: ラベルを付与することでターム単位の一括管理が可能

## UC-GRP-05: グループイベント登録（日付ベース）

- **アクター**: グループ owner / admin
- **主フロー**:
  1. タイトル・日付・種別（event / holiday / examination_period）を入力する
  2. 終日 or 時限指定を選択する
  3. グループイベントとして登録する
- **用途**: 学校行事、試験、休校日などの個別イベント

## UC-GRP-06: グループ一覧・詳細表示

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. 自身が所属するグループ一覧を取得する
  2. グループ詳細（メンバー・予定）を表示する
