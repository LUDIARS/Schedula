# M1: 施設予約 ユースケース

## UC-M1F-01: 施設予約の作成

- **アクター**: 認証済みユーザー
- **事前条件**: グループに所属していること
- **主フロー**:
  1. 予約タイトル・日時・教室・参加者を入力する
  2. 教室の空き状況を確認する
  3. 予約を確定する
  4. カレンダーに予定として自動登録される
- **代替フロー**:
  - 教室が既に使用中の場合、エラーを返す

## UC-M1F-02: 施設予約のキャンセル

- **アクター**: 予約の作成者
- **主フロー**:
  1. 予約を選択してキャンセルする
  2. 関連するカレンダー予定が連動削除される

## UC-M1F-03: 施設予約の一覧表示

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. 教室別・日付別のスケジュールを表示する
  2. 空き状況をグリッド表示で確認する

## UC-M1F-04: 施設の空き状況照会

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. 日付・コマ範囲を指定する
  2. 利用可能な教室の一覧を取得する
