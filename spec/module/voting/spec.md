# Voting（日程調整）仕様

## 概要

投票ベースの日程調整を提供する M6 モジュール。候補日時を設定し、メンバーが ○/△/× で回答する。カレンダー空き状況に基づく自動回答機能あり。

## 機能要件

### 投票イベント

- タイトル・説明・回答期限を設定
- 候補日時をラベルとして複数登録
- ステータス: `open` / `closed`

### 回答方式

| 回答 | 意味 |
|------|------|
| ok (○) | 参加可能 |
| maybe (△) | 未定 |
| ng (×) | 参加不可 |

### 自動回答

- ユーザーのカレンダー空き状況を解析
- 候補日時と照合し、自動で ○/△/× を回答
- `isAutoReply` フラグで手動回答と区別

### 予約プラグイン

- `ReservationPlugin` インターフェースを実装
- プラグイン ID: `voting`
- 予約ランチャーから日程調整を起動可能
