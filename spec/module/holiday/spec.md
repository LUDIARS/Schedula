# Holiday（休日管理）仕様

## 概要

休日・休業期間を管理するモジュール。日本の祝日自動計算、グループ固有の休日、審査会期間などを管理し、スケジュール配置時の休日考慮ユーティリティを提供する。

## 機能要件

### 休日種別

| 種別 | 説明 |
|------|------|
| national_holiday | 日本の祝日 |
| school_holiday | 学校休日 |
| examination_period | 審査会期間 |
| custom | カスタム休日 |

### 日本の祝日自動計算

- ルールベースで1900〜2099年の日本の祝日を計算
- 振替休日・国民の休日にも対応
- DB に一括同期する機能あり

### グループ固有の休日

- グループ ID を指定して休日を登録可能
- グループ ID が null の場合はシステム全体の休日

### スケジュール考慮ユーティリティ

- 指定期間内の営業日計算
- 休日を考慮したスケジュール配置のブロック機能
