# Holiday ユースケース

## UC-HOL-01: 休日の登録

- **アクター**: グループ管理者 / 管理者
- **主フロー**:
  1. 休日名・日付・種別を入力する
  2. 期間休日の場合は終了日も設定する
  3. グループ ID を指定（またはシステム全体）
  4. 繰り返し（none / yearly）を選択する

## UC-HOL-02: 日本の祝日同期

- **アクター**: 管理者
- **主フロー**:
  1. 同期年度を指定する
  2. 祝日計算エンジンで祝日を算出する
  3. DB に一括登録する

## UC-HOL-03: 休日一覧取得

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. 期間・グループ ID を指定する
  2. 該当する休日一覧を取得する

## UC-HOL-04: スケジュール配置時の休日チェック

- **アクター**: システム内部
- **主フロー**:
  1. 配置対象の日付を受け取る
  2. 休日テーブルと照合する
  3. 休日の場合は配置をブロックする
