# M1: 時間割自動生成 ユースケース

## UC-M1S-01: 時間割の自動生成

- **アクター**: 教務管理者
- **事前条件**: カリキュラム・講師・教室・出講可能スロットが登録済み
- **主フロー**:
  1. ターム・配置モード（Pack / Spread）を選択する
  2. 自動生成を実行する
  3. 各カリキュラムが制約を満たすスロットに配置される
  4. 配置結果をグリッド表示で確認する
- **代替フロー**:
  - 配置不可のカリキュラムが存在する場合、警告を表示する

## UC-M1S-02: コマ入れ替え

- **アクター**: 教務管理者
- **事前条件**: 自動生成済みの時間割が存在する
- **主フロー**:
  1. 配置済みのコマをクリックする
  2. 入れ替え可能な先のコマがハイライト表示される
  3. 入れ替え先を選択して確定する
  4. 影響を受ける他のコマが自動的に再配置される
- **制約**: 入れ替えの連鎖は最大2段階まで

## UC-M1S-03: 時間割の確定

- **アクター**: 教務管理者
- **主フロー**:
  1. 配置結果を確認する
  2. 確定操作を行う
  3. M2（データ統合）へエクスポートされる
