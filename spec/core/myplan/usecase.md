# MyPlan ユースケース

## UC-MP-01: マイプラン作成

- **アクター**: 認証済みユーザー
- **主フロー**:
  1. プラン名・パターンタイプ（basic / special）を設定する
  2. 週間スケジュールを曜日ごとに登録する（各エントリに開始時刻・終了時刻・タイトルを設定）
  3. 適用期間（開始日・終了日）を任意で設定する
  4. プランを保存する
- **事後条件**: 対応する個人予定が自動生成される

## UC-MP-02: マイプラン編集

- **アクター**: プランの所有者
- **主フロー**:
  1. 既存プランの内容を変更する
  2. 関連する個人予定が再生成される

## UC-MP-03: マイプラン有効化 / 無効化

- **アクター**: プランの所有者
- **主フロー**:
  1. プランの有効/無効を切り替える
  2. 無効化すると関連する個人予定が非表示になる

## UC-MP-04: グループ向けマイプラン

- **アクター**: グループ管理者
- **主フロー**:
  1. グループIDを指定してマイプランを作成する
  2. グループの空き計算に反映される
